# Data dictionary — Field-level reference

> This document describes the input data fields required by the analysis scripts.
> For the full data dictionary, see the companion data archive at [10.5281/zenodo.21177297](https://doi.org/10.5281/zenodo.21177297).

---

## Input: Grid-level phenology panels

Files: `data/04_Grid_Panels/grid_panel_{maize,wheat,rice}.csv`

Raw 1-km grid-level crop phenology panels, 2000–2019, derived from ChinaCropPhen1km. Each CSV contains pixel-year observations:

| Column | Type | Description |
| --- | --- | --- |
| `crop` | str | Crop name (maize / wheat / rice) |
| `year` | int | Year (2000–2019) |
| `grid_row` | int | Grid row index in the 1-km Albers equal-area grid |
| `grid_col` | int | Grid column index |
| `x_center_m` | float | Albers equal-area projected x-coordinate of pixel centre (m) |
| `y_center_m` | float | Albers equal-area projected y-coordinate of pixel centre (m) |
| `v3_doy` | float | Early-stage phenology DOY (maize: third-leaf; wheat: green-up; rice: transplanting) |
| `he_doy` | float | Heading-stage phenology DOY (maize: tasseling; wheat/rice: heading) |
| `ma_doy` | float | Maturity DOY |

Day-of-year (DOY): Jan 1 = 1.

## Input: Climate data cube

File: `data/05_Climate_Data/china_cube_2000_2019.npz`

NumPy `.npz` archive containing annual extreme-climate index arrays for China at 0.1° resolution, 2000–2019, derived from ERA5-Land daily reanalysis. Arrays (all shape `(20 years, ny_lat, nx_lon)`, dtype float32):

| Key | Description | Unit |
| --- | --- | --- |
| `TXx` | Annual maximum daily maximum temperature | °C |
| `TNn` | Annual minimum daily minimum temperature | °C |
| `PRCPTOT` | Annual total precipitation | mm |
| `CDD` | Consecutive dry days | days |
| `dry_hot` | Dry-hot compound event days per year | days |
| `wet_hot` | Wet-hot compound event days per year | days |
| `dry_cold` | Dry-cold compound event days per year | days |
| `wet_cold` | Wet-cold compound event days per year | days |
| `lat` | Latitude coordinates | degrees |
| `lon` | Longitude coordinates | degrees |
| `years` | Year axis (int32, 2000–2019) | year |

## Output: Analysis panels (generated)

Files: `outputs/panels/panel_{maize,wheat,rice}.parquet` (generated by `build_pheno_climate_panel.py`)

Phenology × climate matched pixel-year long panels. All columns from both input sources merged via nearest-neighbour matching.
